4d-component-7-zip
==================

4D wrapper component to call the 7-zip program

#### Binary version

* macOS

  * ARM: `17.04.arm64_big_sur`
  * x68_64: `17.04.catalina`

#### Notes

To download a homebrew bottle (binary) for M1 on an Intel Catalina

```sh
brew fetch p7zip --bottle-tag=arm64_big_sur
```
